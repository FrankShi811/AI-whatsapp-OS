# AI Sales OS v4.2.0

发布日期：2026-07-24

## 暗色模式高对比

- 全部主题语义色改为动态资源，应用内切换主题后页面、弹窗、新手引导、图表和卡片立即使用当前主题。
- 修复 Dashboard、商机智能、客户智能分析、WhatsApp Inbox、邮件 Inbox、自动化触达、客户编辑和导入窗口中的低对比文字及浅色硬编码背景。
- TextBlock、Label、RadioButton 和 GroupBox 默认继承可读的高对比前景色。
- 回归测试会扫描主题敏感的静态资源、XAML 硬编码颜色，并对主要深色文字组合执行 WCAG AA 4.5:1 对比度校验。

## 更新后快捷方式

- Velopack 安装包继续创建桌面和开始菜单快捷方式。
- 正式安装版每次启动都会检查并修复缺失的桌面与开始菜单快捷方式，因此自动更新后仍可从原入口打开程序。
- 便携版不会擅自在桌面创建快捷方式；完成正式安装后自动进入该保障流程。

## 发布边界

- 本版本仅发布 Windows 中文安装包与 Velopack 自动更新资产。
- 不覆盖或重启开发电脑上的现有 AI Sales OS。
- macOS 构建继续暂停，等待 Apple Developer ID、签名与公证条件恢复。
