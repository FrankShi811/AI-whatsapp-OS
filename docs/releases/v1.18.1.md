# AI Sales OS v1.18.1

## GitHub 自动更新稳定性

- 正式安装版改为直接读取 GitHub Release 中的 `releases.<channel>.json` 静态 Velopack 清单。
- 不再调用匿名 GitHub Releases API，避免共享配额耗尽造成 `403 (rate limit exceeded)`。
- Velopack 仍按照清单内的 SHA-256 校验更新包，下载、安装和重启流程保持不变。
- 便携版也通过静态清单识别最新版本，再下载对应中文 Setup / PKG。

## WhatsApp 最新动态

- 识别 WhatsApp Status / Updates mention 协议，不再把动态链接误当成普通客户私聊。
- 动态仍会显示在会话中，但带有“WhatsApp 最新动态 · 非普通聊天消息”标识。
- 最近 24 小时内的最新动态在对应客户对话顶部显示固定提示，可直接打开动态链接；到期后自动取消置顶。
- 动态不会写入客户最近回复、商机评分、AI 会话助理证据或客户智能分析报告。

## 未读状态

- 新增会话本地已读游标，打开会话即持久化已读时间。
- 返回 WhatsApp Inbox、重新启动或再次同步时，不再用手机端历史同步中的旧未读数量覆盖本地已读结果。
- 新到达的真实消息仍会正常增加未读数。
