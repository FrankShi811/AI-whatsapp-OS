# AI Sales OS v1.17.2

## 正式安装通道修复

- 便携版匹配 GitHub Release 安装资产时忽略空格、点号和连字符，兼容 GitHub 实际保存的 `AI.Sales.OS.Setup.exe` 与 macOS PKG 名称。
- Windows 便携版可下载正式 Setup；macOS 会按 Apple Silicon / Intel 架构选择对应中文 PKG。
- 完成一次正式安装后，后续版本继续使用 Velopack 标准自动更新。

## AI 分析稳定性

- AI Provider 的结构化输出支持 camelCase、snake_case、代码围栏、尾随逗号、数字字符串及 JSON 前后说明文字。
- 首次输出不合规时携带校验原因自动纠正重试一次。
- 商机智能把资料不足视为可完成的低置信度 D / 0 结果；格式异常、限流、网络或配置问题才进入明确的可重试状态。
- 客户智能分析在 AI 阶段仍异常时，继续基于 CRM、WhatsApp、邮件、自动化触达、商机历史与客户轨迹中的已核验资料生成当前版本，并标记信息缺口。
- 后续新增资料后可再次生成新版本，不覆盖历史报告或 CRM 原始数据。

## 新手引导与数据边界

- Dashboard、商机智能、客户列表、WhatsApp Inbox、邮件 Inbox、自动化触达、客户智能分析和 API 设置均有独立首次蒙层与常驻使用手册。
- 更新不覆盖 `%LOCALAPPDATA%\WAFlow\waflow.db`、账号、会话、邮件、API 凭据、自动化任务或报告历史。
- 本版本仅通过 GitHub Release 发布，不替换、关闭或启动用户电脑当前的 `AI Sales OS.exe`。
