# AI Sales OS v2.0.0

## AI 原生桌面设计系统正式版

- 完成 Phase 1 UI Audit，逐页确认 Dashboard、商机智能、客户列表、WhatsApp Inbox、邮件 Inbox、自动化触达、客户智能分析和 API 设置的信息架构与工作流。
- 完成 Phase 2 Figma Design System：120 个变量、10 套文字样式、4 套效果和 7 个核心 AI 组件。
- WPF 全局语义 token 与 Figma 浅色/深色主题对齐，统一执行绿、AI 紫、处理青、状态色、ABCD 等级色与 WhatsApp 消息色。
- 统一页面标题、正文、标签、指标、按钮、输入框、表格、卡片、圆角与深度尺度。
- 新增可复用的 Holographic Card、AI Confidence Meter、Reasoning Step、Priority Card、Message Bubble 和 Workflow Node 样式。
- AI Score Ring 增加短时缓动并尊重 Windows 减少动画设置；Dashboard 与商机智能使用相同等级颜色契约。

## 功能与数据边界

- 不修改 CRM、WhatsApp、邮件、AI Provider、自动化触达、SQLite 和自动更新业务逻辑。
- 升级继续保留 `%LOCALAPPDATA%\WAFlow\waflow.db`、WhatsApp / 邮件账号凭据和全部用户配置。
- Windows 与 macOS 继续使用 GitHub Release / Velopack 自动更新通道；只有用户确认后才安装并重启。
