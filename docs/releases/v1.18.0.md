# AI Sales OS v1.18.0

## WhatsApp AI 会话助理

- WhatsApp Inbox 输入区新增“✦ AI”按钮，使用 API 对接中已经选择的 DeepSeek / OpenAI 兼容模型。
- AI 读取当前会话上下文，但客户需求与 CRM 字段更新只能引用客户发送的白色气泡原话。
- 生成可编辑回复，并以中文整理需求摘要、意向、采购信号、风险和下一步动作。
- 发送前显示完整预览：用户可修改回复、选择要同步的客户维度，也可只把草稿填入输入框。
- 选择“发送并同步客户资料”后，AI 摘要和勾选字段进入统一客户档案；未关联联系人会在确认后创建 D 级 / 0 分客户。
- AI 提取不会直接修改客户评分；新信息会进入商机智能分析队列，只有正式结构化评分成功后才更新等级和 Dashboard。
- 所有生成、字段证据和同步动作写入客户审计轨迹，可供后续客户智能报告读取。

## 安全边界

- AI 不会根据销售人员自己的绿色气泡反推客户需求。
- 没有客户原话证据的字段更新会被结构校验拒绝。
- 姓名、电话、WhatsApp 号码、负责人、退订状态和 AI 分数不允许由会话助理自动修改。
- WhatsApp 发送成功仍以服务端回执为准；状态不确定时不会自动重复发送。

## 新手引导

- WhatsApp Inbox 独立使用手册新增 AI 会话助理的功能介绍、操作步骤和数据写入边界。
