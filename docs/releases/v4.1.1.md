# AI Sales OS v4.1.1

发布日期：2026-07-24  
变更规模：PATCH

## Customer Intelligence 摘要展示修复

- WhatsApp Inbox 的 AI Sales Brief 取消客户画像固定高度与省略号裁切，完整文本会按侧栏宽度自然换行。
- AI 画像与下一步行动改为自适应两行布局，互不覆盖，长内容由客户侧栏统一滚动查看。
- 新增 UI 回归契约，禁止该区域重新引入固定高度、字符省略或固定像素叠放。

## 版本号规范

- 正式采用 `MAJOR.MINOR.PATCH` 三段式版本规则。
- 大规模或不兼容升级递增 `MAJOR`，中等规模兼容功能递增 `MINOR`，小型修复递增 `PATCH`。
- 每次发布按全部变更中的最高影响级别自动判定，并同步程序、版本历史、发布说明、GitHub Release 与 Velopack 资产。

## 兼容性与发布边界

- 不修改客户、WhatsApp 账号、Customer Brain、商机、邮件、自动化或本地配置数据。
- 仅通过 GitHub Release 发布 Windows 中文安装包和自动更新资产。
- 不覆盖或重启本机正式程序；macOS 构建继续暂停。
