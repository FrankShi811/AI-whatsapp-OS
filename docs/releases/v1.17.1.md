# AI Sales OS v1.17.1

## AI 结构化输出修复

- 兼容 AI Provider 返回的 camelCase、snake_case、代码围栏、尾随逗号和 JSON 前后说明文字。
- 结构化结果首次不合规时自动携带校验提示纠正并重试一次。
- Lead Intelligence 六维评分只保留具有可核验证据的得分；缺少证据的维度归零并明确说明，不使用本地关键词补分。
- 商机列表的失败状态区分 AI 格式异常、API 限流、网络异常、AI 配置异常和用户停止。

## 当前资料也能生成客户情报报告

- 客户情报继续读取 CRM、WhatsApp、邮件、Lead Intelligence、自动化触达和客户轨迹的全部当前可用数据。
- 单个 AI 阶段仍无法通过结构校验时，不再让整份报告失败；系统以已核验事实生成安全降级版本，并在页面标出信息缺口。
- 缺失资料不会被编造成事实，成交概率不会在证据不足时被本地规则擅自推高。
- 后续新增回复、字段或跟进记录后，可再次点击生成，形成包含新快照的新版本。

## 更新与新手引导

- 便携版可检查 GitHub Release 并下载适合当前 Windows 或 macOS 架构的正式安装包。
- Windows 用户完成一次 `AI Sales OS Setup.exe` 正式安装后，后续版本进入 Velopack 标准自动更新通道。
- 校验 Dashboard、商机智能、客户列表、WhatsApp Inbox、邮件 Inbox、自动化触达、客户智能分析和 API 设置八个板块均有独立首次蒙层与常驻使用手册。

## 数据边界

- 更新不覆盖 `%LOCALAPPDATA%\WAFlow\waflow.db`、客户数据、WhatsApp 账号、邮件账号、API 凭据、自动化任务或历史报告。
- 本版本只通过 GitHub Release 发布；开发流程不替换、关闭或启动用户电脑当前的 `AI Sales OS.exe`。
