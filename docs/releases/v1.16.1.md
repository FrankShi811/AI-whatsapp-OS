# AI Sales OS 1.16.1

## 自动更新验证

- 完成 GitHub Release 首次发布与后续版本更新链路验证。
- Windows 使用 `win` 通道，Apple Silicon 使用 `osx-arm64` 通道，Intel Mac 使用 `osx-x64` 通道。
- 程序启动后检查最新稳定版本，后台下载完成后由用户在版本中心确认安装并重启。

## 稳定性

- 修复 GitHub Actions 中 WhatsApp Bridge 的 pnpm 依赖安装。
- 修复首次发布不存在历史增量包时的错误退出。
- 修复 macOS Velopack 跨平台参数兼容问题。
- 内置 Noto Sans CJK SC 中文字体，确保客户智能报告在不同 Windows 环境均可导出 PDF。

## 数据安全

- 更新只替换应用安装目录，不覆盖 `%LOCALAPPDATA%\WAFlow\waflow.db`。
- 客户数据、WhatsApp 账号、会话文件、API 配置、群发任务与历史报告继续保留。
