# AI Sales OS v3.0.0

发布日期：2026-07-23

## 个人 AI 销售员工

- 正式交付 AI Sales OS 3.0 第一阶段能力，产品继续定位为“每一位销售人员的个人 AI 销售员工”。
- 保留当前正式版 UI，不改变用户已经熟悉的 Dashboard、商机智能、客户列表、WhatsApp、邮件、自动化和客户分析工作流。
- 不引入团队权限、共享收件箱、多人协作、主管看板或团队路由。

## Customer Brain 分阶段 AI 管线

- 新增“资料整理 → 客户理解 → 商机评估 → 销售建议”的分阶段运行链路。
- 每一步保存输入快照、Provider、模型、结构化中间结果、运行状态、错误和可重试状态。
- 新资料到达后旧结论会标记为需要重新分析，但不会删除历史版本。
- AI 失败时保留最近一次有效判断，不使用本地关键词或规则伪造成功结果。

## 商机决策与 Customer 360

- 新增购买概率、AI 置信度、建议阶段、证据、下一步行动和信息缺口。
- 客户生命周期扩展至需求确认、报价与复购等个人销售常用阶段。
- 新增跟进任务、客户事件时间线和建议历史，销售动作可完成、延期并记录结果。
- 客户编辑页加入 Customer 360 与“AI 分析并生成行动”，统一呈现事实、判断、建议、任务和客户历史。
- AI 结论只提供建议，不擅自覆盖人工维护的 CRM 信息。

## 发布与数据边界

- 本版本通过 GitHub Release 发布 Windows 中文安装包、Velopack 全量/增量包和自动更新清单。
- 更新继续保留 `%LOCALAPPDATA%\WAFlow\waflow.db`、WhatsApp 登录会话、邮件账号、客户资料和本地配置。
- 不主动安装、覆盖或重启用户本机的正式程序，由软件自动检测更新并由用户点击安装。
- macOS 新版本构建继续暂停，等待 Developer ID、签名与公证条件具备后再恢复。
