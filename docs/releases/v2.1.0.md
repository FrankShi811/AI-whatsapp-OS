# AI Sales OS v2.1.0

发布日期：2026-07-23

## Stitch 视觉方向落地

- 审核用户从 Google Stitch 项目导出的 12 套 Auralis 设计变体。
- 选定 `auralis_ai_voice_infrastructure_refined_alignment_transparency` 作为主参考。
- 将参考稿转译为原生 WPF 设计组件，不嵌入 HTML、Tailwind 或 WebView。
- 新增半透明卡片、环境光页头、AI 决策玻璃卡和高层指标卡。
- Dashboard、商机智能和客户智能分析统一为更清晰、更克制的 AI 原生企业界面。

## 性能与业务边界

- 环境光为静态 WPF Brush，不使用持续循环装饰动画。
- 阴影仅用于高层容器和 AI 重点区域。
- CRM、WhatsApp、邮件、AI Provider、自动化、SQLite 和更新逻辑保持不变。
- 已在隔离数据库下完成关键页面视觉冒烟，不触碰用户当前安装和业务数据。

## 发布策略

- Windows 中文 Setup、Velopack 包与更新清单继续通过 GitHub Release 发布。
- macOS 新版本构建默认暂停。
- 只有获得有效 Developer ID、签名/公证条件并收到用户明确恢复通知后，才重新启用 Mac GitHub Actions。
