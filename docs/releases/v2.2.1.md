# AI Sales OS v2.2.1

发布日期：2026-07-23

## 深色主题与交互可读性

- 深色模式升级为高对比海军蓝与石墨色体系，统一修复标题、正文、卡片、表单、图表、弹窗和状态文本的对比度。
- 邮件 Inbox 的 Customer Intelligence 改为可收起抽屉，展开和收起时内容区域自动调整。
- 八个核心板块的新手引导按“板块 + 内容版本”记录；程序升级后，未变化的教程不会重复弹出。

## AI Provider 与商机智能

- 新增 DeepSeek、OpenAI、Google Gemini、xAI、OpenRouter、Groq、Together AI、Mistral、Qwen、Moonshot/Kimi、智谱 GLM、SiliconFlow 和自定义兼容接口。
- 选择 Provider 后自动填入默认 Base URL，并保留手动修改能力；API Key 通过真实模型接口验证，可实时拉取账户可用模型。
- 已配置的 Provider、模型与连接状态会持久保存并展示。
- 人工修改客户商机阶段后自动锁定，AI 后续分析不再覆盖；用户可明确解除锁定。
- 批量分析支持中断续跑，只处理未完成客户，避免从头分析和重复消耗 Token。
- 批量分析与更新安装取消多余的二次确认，点击后直接执行。

## WhatsApp Inbox 稳定性

- 登录失效或远端退出后清理无效认证状态，再次连接时可重新生成二维码扫码。
- 已读游标只向前推进，切换页面或后台同步不会重新恢复已清除的未读气泡。
- WhatsApp 动态只保留会话顶部的 24 小时置顶提醒，不再重复显示为普通聊天气泡，也不计入未读数。
- WhatsApp Bridge 改为按内容哈希生成并存运行文件，更新时不再强制覆盖正在使用的旧桥接程序。

## 发布与数据边界

- 本版本通过 GitHub Release 发布 Windows 中文安装包、Velopack 全量/增量包和自动更新清单。
- 更新继续保留 `%LOCALAPPDATA%\WAFlow\waflow.db`、WhatsApp 登录会话、邮件账户和本地配置。
- macOS 新版本构建继续暂停，等待 Developer ID、签名与公证条件恢复后再启用。
