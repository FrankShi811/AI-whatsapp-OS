# AI Sales OS v2.2.0

发布日期：2026-07-23

## 正式 UI 恢复

- 恢复并继续使用 `v2.1.0` 已发布的正式 UI。
- 撤销未获验收的 Aurora Nexus 视觉实验，不改变现有页面结构、操作习惯和数据密度。
- 保留 Customer Brain 在商机智能与 WhatsApp 客户侧栏中的最小状态呈现，不引入新的全局视觉主题。

## Customer Brain 跨渠道客户大脑

- 新增统一客户情报快照，整合 CRM、自定义字段、WhatsApp 会话、邮件、自动化触达、Lead Intelligence 与客户历史轨迹。
- 情报内容固定区分事实、AI 判断、销售建议和信息缺口，并记录证据来源、覆盖率、版本与生成时间。
- 新增客户大脑、推荐历史、行为时间线、销售动作日志与 AI 学习反馈的本地数据库表。
- 同一客户重复刷新采用幂等版本策略；只有来源数据发生变化时才生成新快照。
- 商机智能和 WhatsApp 客户侧栏可显示证据覆盖、事实/判断/缺口数量、客户摘要与下一步行动。

## 兼容性与数据安全

- 采用仅新增表的 SQLite 迁移，不删除、不覆盖现有客户、WhatsApp 账号、邮件、配置或分析数据。
- CRM、WhatsApp 连接、发送、自动化、AI Provider 与 GitHub 自动更新逻辑保持原有行为。
- 本版本仅通过 GitHub Release 发布 Windows 中文安装包、Velopack 包和更新清单。
- macOS 新版本构建继续暂停；只有具备 Developer ID、签名/公证条件并收到明确恢复通知后才重新启用。
- 开发发布流程不覆盖、关闭或启动用户电脑当前的 `AI Sales OS.exe`；由用户在版本中心手动确认安装。
