# AI Sales OS 1.16.0

## 新增

- 启动时自动检查 GitHub Release 最新稳定版本。
- 发现更新后在后台自动下载，左下角版本入口实时显示下载状态。
- 版本中心显示当前版本、最新版本、远程更新日志和更新错误。
- 用户点击“安装更新并重启”后，由 Velopack 完成替换并重新打开 AI Sales OS。

## 数据安全

- 更新只替换应用安装目录，不覆盖 `%LOCALAPPDATA%\WAFlow\waflow.db`。
- WhatsApp 登录会话、媒体、运行时文件和 Windows 凭据继续保存在原来的本地位置。
- 现有客户、聊天、Campaign、API 配置和账号状态无需迁移。

## 交付

- Windows x64 使用固定安装入口 `AI Sales OS Setup.exe`。
- 根目录仍只保留固定名称 `AI Sales OS.exe`，后续构建直接覆盖。
