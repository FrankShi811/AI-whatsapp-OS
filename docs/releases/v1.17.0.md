# AI Sales OS v1.17.0

## 邮件 Inbox

- 在 Customer Operations 下新增邮件 Inbox。
- 支持 Gmail、Outlook / Microsoft 365、Yahoo、iCloud 与自定义 IMAP / SMTP 账户。
- 邮箱密码或应用专用密码只写入系统凭据存储，不写入 SQLite、日志或 Git 仓库。
- 支持收件箱同步、会话列表、邮件正文、直接回复、未读状态与本地历史保存。
- 邮箱地址会自动匹配客户列表；未匹配联系人可从右侧客户栏创建并同步到客户列表和商机智能。

## 多渠道自动化触达

- 原 WhatsApp 自动化群发升级为 WhatsApp / 邮件双渠道任务。
- 邮件任务支持主题模板、正文模板、CRM 动态字段、即时或定时启动、逐人间隔与每日上限。
- 任务发送历史新增渠道列，WhatsApp 与邮件分别记录成功、失败、等待、跳过和错误原因。
- 邮件成功表示 SMTP 服务器已经接受；后续退信仍以收件服务器返回为准。

## WhatsApp 号码与发送修复

- 修正“本地未同步到联系人就等于号码不存在”的错误判断。联系人缓存缺失只作为诊断信息，不再阻止发送。
- 发送时优先使用当前会话、联系人缓存或 WhatsApp 返回的真实 JID / LID 映射，最后才按号码构造目标。
- WhatsApp 异步失败回执继续覆盖早期乐观状态，历史任务不会把真实失败统计为成功。
- 失败记录增加回执上下文，便于继续判断账号连接、JID 映射或 WhatsApp 服务端拒绝原因。

## 数据与更新边界

- 客户情报报告快照新增邮件历史和邮件自动化触达数据。
- 更新不会覆盖 `%LOCALAPPDATA%\WAFlow\waflow.db`、WhatsApp 会话、邮件历史、API 配置或客户报告。
- 本版本只通过 GitHub Release 发布。开发流程不替换、关闭或启动用户电脑当前的 `AI Sales OS.exe`；用户在程序版本中心手动确认安装更新。
